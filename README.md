# Super Mario Bros. Wonder - Speedrun Practice Mod

This is a modification of Super Mario Bros. Wonder created for speedrunners of the game that want to practice strategies more efficiently than what the vanilla game offers.

It is currently a Work-In-Progress, focusing on changes that optimize practice for the game's most popular category: **Any% No Major Glitches**. Changes made for the other main categories, **Any% & 100%**, will be the main focus after NMG changes are completed. 

---
## Installation

You will need **version 1.0.0 of Wonder** to use this modification, as well as either a modified Nintendo Switch running Atmosphere Custom Firmware or a Nintendo Switch Emulator (such as Ryujinx). 

If you don't have a modified Switch, **check out [this guide](https://switch.hacks.guide)** to see if your Switch is compatible, or what you need to look for when buying another Switch.

If you are using a modified Switch and *currently use a version of Wonder that is above v1.0.0*, **use [this tutorial](https://www.youtube.com/watch?v=mtZ_s4zwVJs) to downgrade your copy!**

## Changes

If a level isn't listed here, it either hasn't been changed yet, or is not significant enough to have any changes to it. This is the case for "Break Time!" and "Badge Challenge" levels. 

- General
	- Custom Title Screen Background
	- An in-game timer is included in *every level*
		- This includes "Break Time!" and "Badge Challenge" levels that otherwise don't have any changes applied!
- World 1
	- *Welcome to the Flower Kingdom! -* Starting pipe sends to section before Wonder Flower
	- *Piranha Plants on Parade -* Starting pipe sends to note-block section before Secret Exit pipe
	- *Scram, Skedaddlers! -* Starting pipe sends to section before grabbing shell for throwing at 3rd 10 coin, pipe after shell throw sends back to previous pipe
	- *Here Comes the Hoppos -* Starting pipe sends to section before hitting shell for activating 3rd 10 coin, pipe right before Wonder Flower sends back to previous pipe
	- *Sproings in the Twilight Forest -* Starting pipe sends to section before Wonder Flower
	- *Cosmic Hoppos -* ***(SUBJECT TO CHANGE)***
	- *Pipe-Rock Plateau Palace -* First pipe sends to section before Checkpoint Flag, second pipes sends to pipe before Wonder Flower, third pipe sends to Bowser Jr. boss fight
	- *KO Arena: Pipe-Rock Rumble -* Added pipes to send to each arena room, Elephant Fruit placed on spawn
- World 2

	- *Condarts Away! -* Starting pipe sends to section before Wonder Flower (Note: Cycle will be off from RTA, use for practicing Wonder Flower entry)
